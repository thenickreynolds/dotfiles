# Example Claude Command

This is an example slash command. To use it, type `/example` in Claude Code.

Commands are markdown files that provide context and instructions to Claude.

## Usage

Add your own commands by creating `.md` files in this directory.

For example:
- `review.md` - Code review instructions
- `test.md` - Test writing guidelines
- `commit.md` - Commit message conventions

## Template

```markdown
# Command Name

Brief description of what this command does.

## Instructions

1. Step one
2. Step two
3. Step three

## Context

Any additional context Claude should know.
```
